## 📚 Essential Python Books

### For Beginners

**"Python Crash Course" by Eric Matthes** (3rd Edition, 2023)
- Hands-on, project-based introduction
- First half: Python fundamentals
- Second half: Three major projects

**"Automate the Boring Stuff with Python" by Al Sweigart**
- Practical programming for beginners
- [Free online version](https://automatetheboringstuff.com/)
- Real-world automation tasks

### For Intermediate/Advanced

**"Fluent Python" by Luciano Ramalho** (2nd Edition, 2022)
- Deep dive into Python's features
- Pythonic idioms and best practices
- Essential for writing professional Python code

**"Effective Python" by Brett Slatkin** (2nd Edition, 2019)
- 90 specific ways to write better Python
- Best practices and common pitfalls
- Intermediate to advanced level

**"Python Cookbook" by David Beazley and Brian K. Jones**
- Recipes for common Python tasks
- Advanced techniques and patterns
- Great reference guide
