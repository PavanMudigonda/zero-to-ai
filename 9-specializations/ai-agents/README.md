# AI Agents Specialization

## 🤖 Overview

Build autonomous systems that can use tools and collaborate!

**Time:** 2-3 months | 150-200 hours  
**Prerequisites:** Phases 1-8 complete

- [ ] Agent evaluation and safety
- [ ] Workflow automation
