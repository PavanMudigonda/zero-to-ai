## 🎓 What You'll Master

### Foundations
- **Mathematics**: Linear algebra, calculus, probability, optimization
- **Python**: NumPy, pandas, matplotlib, scikit-learn
- **Tokenization**: tiktoken, HuggingFace tokenizers, SentencePiece

### Core ML/DL
- **Neural Networks**: Build from scratch, backpropagation, PyTorch
- **Transformers**: Attention mechanisms, BERT, GPT, fine-tuning
- **Embeddings**: Word2Vec, sentence embeddings, semantic search

### Vector Databases
- **Chroma**: Lightweight, developer-friendly
- **Qdrant**: Production-ready, filtering
- **Weaviate**: GraphQL, hybrid search
- **Milvus**: Scalable, enterprise
- **Aurora PostgreSQL**: pgvector, AWS-native

### Production Skills
- RAG (Retrieval-Augmented Generation)
- Semantic search & similarity
- Model deployment & optimization
- Cloud ML (AWS, Azure)

## 💻 Prerequisites

- **Python** - Basic programming (loops, functions, classes)
- **Math** - High school algebra (Khan Academy for review)
- **Time** - 10-15 hours/week for 6-12 months
