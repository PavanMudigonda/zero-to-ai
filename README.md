## 🚀 Quick Start

### Prerequisites
- Basic Python knowledge (or start with [0-python/](0-python/))
- High school mathematics
- A computer (GPU helpful but not required initially)
- 10-15 hours/week

### Installation (Choose One)

#### Option 1: UV - Fast & Recommended
```bash
# Clone the repository
git clone https://github.com/PavanMudigondaTR/zero-to-ai.git
cd zero-to-ai

# Install dependencies with UV (fastest!)
./install_dependencies.sh

# Start learning
jupyter notebook
```

#### Option 2: Conda - Best for ML/Data Science
```bash
# Clone the repository
git clone https://github.com/PavanMudigondaTR/zero-to-ai.git
cd zero-to-ai

# Create conda environment
conda env create -f environment.yml
conda activate aiml-learning

# Start learning
jupyter notebook
```

## 🗺️ Learning Paths

Choose your path based on your goals and background:

### 🎯 Path 1: Complete AI/ML Engineer (6-12 months)
**For**: Career switchers, serious learners wanting comprehensive knowledge

```
Phase 0-1: Foundations (2 months)
    ↓
Phase 2-4: Core Concepts (3 months)
    ↓
Phase 5-7: Deep Learning (3 months)
    ↓
Phase 8-13: Advanced & Production (4 months)
```

**Outcomes**: Job-ready AI/ML engineer with portfolio projects

### ⚡ Path 2: LLM/GenAI Specialist (3-4 months)
**For**: Developers wanting to build with LLMs quickly

```
Skip to: Phase 3 (Tokenization)
    ↓
Phase 4: Embeddings
    ↓
Phase 7: RAG Systems
    ↓
Phase 10-11: Prompt Engineering & Fine-tuning
```

**Outcomes**: Build production LLM applications, RAG systems

### 🎨 Path 3: Specific Specialization (2-3 months)
**For**: Focused learning in one area

**Options:**
- **Computer Vision**: Phase 5 → Phase 9 (Vision) → Phase 12 (Multimodal)
- **NLP**: Phase 3 → Phase 4 → Phase 9 (NLP) → Phase 10
- **AI Agents**: Phase 7 → Phase 9 (Agents) → Phase 10
- **MLOps**: Phase 1 → Phase 5 → Phase 8

### 📚 Path 4: Academic Preparation (4-6 months)
**For**: Students preparing for advanced degrees or research

```
Phase 0-2: Math-heavy foundations
    ↓
Phase 5: Neural Networks (theoretical)
    ↓
Read research papers in specialization area
    ↓
Original research project
```
